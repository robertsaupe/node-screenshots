## Installing

### Requirements

- [Install Node.js](https://nodejs.org)

### Environment

```bash
git clone https://github.com/robertsaupe/node-screenshots.git
cd node-screenshots/
npm i
```

## Getting started

modify example.js or create your own

then simple run:

```bash
node example.js
```

you can find screenshots in ./screenshots/
